Hi, Welcome To The Blazapp!

Register as a student here if you have no account so that you can request for a book from the library. You need to return the book within 14 days or else you will be subject to a fine according to how long you take to return the book. A student will be given a Library ID upon registration with which the borrowed book will be attached and he or she will be allowed one book at a time. Upon return of the book, the student will be allowed to request for another book.
